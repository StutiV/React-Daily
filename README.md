
Basic counter app with no view layer (run on the terminal using babel-cli) to learn the basics of Redux.
(creating a reducer, a store and testing the store)

### Running the app

1. Ensure you have `npm` installed.

Follow the instructions for your platform [here](https://github.com/npm/npm).

2. Install all dependencies:

````
npm install
````

3. We execute the files in the terminal using `babel-node` which comes with `babel-cli`:

````
./node_modules/.bin/babel-node app.js
````
